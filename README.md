# Marketing AI Skills

A complete, step-by-step web guide to building a **5-agent, 12-skill autonomous AI marketing team** inside Claude Code.

🌐 **Live site:** https://azizsaif1967-afk.github.io/marketing-ai-skills/
🎥 **Source video:** [Claude Code: Build Your Full AI Marketing Team](https://www.youtube.com/watch?v=yLXLHnD4fco&t=10s) by Grace Leung

## What's inside

This single-page HTML guide consolidates Grace Leung's YouTube walkthrough and the official screen-by-screen PDF SOP into one verified, sequential document.

**12 sections:**

1. Overview — what you're building
2. 4-step team design framework
3. Install Claude Code (VS Code / CLI / Cursor)
4. Project folder structure
5. Build skills — 2 full examples + 36-skill library
6. Build agents — Data Analyst, Content Creator + pattern for 3 more
7. Agent routing with `CLAUDE.md`
8. Notion task board + Nano Banana MCP remote control
9. Screen-by-screen video walkthrough (with timestamps)
10. 30-minute install & run SOP
11. Full skills reference (36 skills)
12. Resources & links
